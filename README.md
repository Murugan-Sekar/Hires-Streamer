# Hi-Res Streamer

Hi-Res Streamer is a Hi-Res lossless music streaming player that supports both online streaming and offline playback.

The application can stream playlists, albums, tracks, and radio using Spotify URLs while also playing local high-quality audio files such as FLAC.

## Features

• Stream music using Spotify playlist / album / radio URLs  
• Play local audio files from device storage  
• Hi-Res and lossless audio playback support  
• Smooth and lightweight UI  
• Built using Flutter, Kotlin, and Go Mobile

## Tech Stack

- Flutter
- Kotlin (Android)
- Go Mobile backend
- Spotify URL streaming

## Installation

1. Clone the repository
   git clone https://github.com/Murugan-Sekar/Hires-Streamer.git

2. Install dependencies
   flutter pub get

3. Build Go backend
   gomobile bind -target=android -androidapi 21 -o gobackend.aar

4. Run the app
   flutter run
